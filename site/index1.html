How to push a file in GitHub

We face problems to learn git initially so here are few steps to push a file in the repository. I hope it will be helpful for beginners. Help me look into these steps whenever I need it.

  1. git clone < URL.git>
  2. cd URL,eg:coursera/
  3. git status
  4. mkdir ....
  5. cd ..../
  6. git add.
  7. git commit -m "Message"
  8. git push
